The Pros and Cons of HTML

The Pros

  • It is easy
  • It is fun
  • Allows great products to be created
  • It is very easy to learn
  • HTML loads fast

The Cons

  • HTML is very time consuming
  • Takes a lot of typing
  • Can take a long time to find a bug in a big website
  • Less secure. Open to SQL injections. HTML5 is the new more secure version (HTML itself does not have specifing security features and can lead to data hacking.)
  • You cannot create games using HTML. You need to use JavaScript instead.
